# Student Management System

A simple Python command-line application for managing students.

## Features
- Add a student
- View all students
- Search by student ID
- Update student information
- Delete a student
- Save data to students.json
- Load existing data automatically when the program starts

## Run the program
```bash
d:/Programming/Innovera/task_04/.venv/Scripts/python.exe student_manager.py
```

d:/Programming/Innovera/task_04/.venv/Scripts/python.exe -m unittest discover -s tests -v
